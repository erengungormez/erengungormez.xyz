KelimeHane

Developed with the Unity game engine, KelimeHane is written in the C# language. KelimeHane is a word game where you try to find the correct word by entering letters on the keyboard into the blanks. It becomes much more enjoyable with levels, scores, and jokers

Stack Craft

Stack Craft, developed with the Unity game engine using the C# language, offers a unique gameplay experience reminiscent of Stack Ball but set in the Minecraft universe. This hypercasual game challenges players to guide the ball through obstacles, aiming to reach the bottom. With an evolving structure and numerous levels, the game becomes increasingly challenging as players progress, adding to the excitement of the journey.
